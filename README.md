The snake game

Installation

    Linux which uses apt (Debian, Ubuntu):

        apt install python3

        apt install python3-tk

        pip3 install pygame

        git clone https://github.com/Ovod18/snake

Usage

    python3 launcher.py

Dependencies

    Python >= 3.9
    Tkinter >= 8.6
    Pygame >= 2.1.2

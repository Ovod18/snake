The snake game

Installation

    Linux which uses apt (Debian, Ubuntu):

        sudo apt install python3

        pip install tkinter

        pip install pygame

        git clone https://github.com/Ovod18/snake

Usage

    python3 launcher.py

Dependencies

    Python >= 3.10
    Tkinter >= 8.6
    Pygame >= 2.1.2
